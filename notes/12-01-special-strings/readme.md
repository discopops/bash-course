# Special Strings

```
echo 'hello\nworld'

echo -e 'hello\nworld'

msg='hello\nworld'
echo "$msg"
msg=$'hello\nworld'
echo "$msg"

echo $'foo\nbar'
echo $'foo\tbar'
echo $'foo\vbar'
echo $'foo\b\b\bbar'
```
